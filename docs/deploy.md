# Deploy - GitHub Pages

## Objetivo

Registrar o processo de publicação da aplicação VeritasIA utilizando GitHub Pages.

## Estratégia

O frontend da aplicação está localizado na pasta `Frontend`.

A publicação será feita a partir da branch responsável pelo frontend, mantendo o histórico de desenvolvimento separado da branch principal.

## Passos planejados

1. Configurar o projeto React/Vite para funcionar no GitHub Pages.
2. Ajustar o caminho base da aplicação no `vite.config.js`.
3. Gerar a versão de produção com `npm run build`.
4. Publicar a pasta gerada no GitHub Pages.
5. Registrar no README o link de acesso da aplicação.
