# Configuração GitHub Pages

## Objetivo

Publicar o frontend da aplicação VeritasIA utilizando GitHub Pages.

## Ferramentas

- Vite
- gh-pages

## Configurações realizadas

- Definição do `base` no `vite.config.js`
- Adição dos scripts de deploy no `package.json`
- Instalação da dependência `gh-pages`

## Próximo passo

Executar o build da aplicação e publicar a pasta `dist`.
