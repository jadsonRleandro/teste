# Autenticação - Frontend

## Objetivo

Criar as telas iniciais de Login e Cadastro da aplicação VeritasIA.

## Funcionalidades planejadas

- Tela de Login
- Tela de Cadastro
- Navegação entre Login e Cadastro
- Integração futura com API de autenticação
- Possibilidade de acessar o Dashboard sem cadastro

## Decisão de fluxo

A aplicação abre diretamente no Dashboard.  
Um modal inicial sugere que o usuário faça login ou cadastro para receber notificações, mas permite continuar sem conta.
